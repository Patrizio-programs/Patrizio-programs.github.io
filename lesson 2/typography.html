Typography

Headings

All heading on the website will be in Alegreya Sans SC. This covers all H1, H2 and H3 tags as well as all links in the navigation. This font was selected because it is a very bold and thick font that is good for headings.




Content

The content on the page will be in Biryani. This will include the P (paragraph) tags. This font was chosen because of it's light yet crisp look.
